# Source Code

This directory contains reusable components used across notebooks.

## Modules

- `pipeline.py`  
  Data preprocessing utilities (feature engineering, cleaning, seasonal labeling).

- `gmm_utils.py`  
  Utilities for fitting and evaluating Gaussian Mixture Models.

## Notes

The notebooks contain the primary analysis workflow. This directory provides shared functionality to avoid duplication, but is intentionally lightweight to preserve transparency and ease of debugging.
