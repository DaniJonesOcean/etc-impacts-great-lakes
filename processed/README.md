# Processed Data

This directory contains intermediate and final data products generated by the analysis notebooks. These files are used to reproduce figures and results in the manuscript.

## Storm datasets

- `cfsr_features.csv`  
  Feature table derived from the CFSR dataset.

- `era5_features.csv`  
  Feature table derived from the ERA5 dataset.

- `cfsr_storms_labeled_k2.csv`  
  CFSR storms with cluster assignments (K = 2) and storm-type labels (early-entry vs late-entry).

- `era5_storms_labeled_k2.csv`  
  ERA5 storms with cluster assignments (K = 2) and storm-type labels.

## Model artifacts

- `final_gmm_k2.joblib`  
  Fitted Gaussian Mixture Model (K = 2).

- `scaler.joblib`  
  StandardScaler used to normalize clustering features.

- `storm_preprocessor.joblib`  
  Preprocessing pipeline applied before clustering.

- `impact_transformer.joblib`  
  Transformation pipeline used for impact variables.

## Thresholds and diagnostics

- `precip_extreme_thresholds.csv`  
  Thresholds used to define extreme precipitation events.

- `evap_extreme_thresholds.csv`  
  Thresholds used to define extreme evaporation events.

## Risk metrics

- `rr_df_all_seasons.csv`  
  Risk ratio and risk difference metrics aggregated across all seasons.

- `rr_df_by_season.csv`  
  Risk metrics computed separately by season.

## Storm statistics

- `storm_counts_by_month_and_type.csv`  
- `storm_counts_by_season_and_type.csv`  
- `storm_fraction_by_month_and_type.csv`  
- `storm_fraction_by_season_and_type.csv`  

Counts and fractions of storms by type and time period.

## Metadata

- `features.json`  
  List of features used in clustering.
